NAME

oltpcheckkey − Check if keys exist in the repository

SYNOPSIS

oltpcheckkey key ...

USAGE

The oltpcheckkey command that take one or more keys as arguments and runs a query to see if these keys already exist in the repository. If they do, the latest entry with that key will be shown.

You typically use this before running oltpscale which requires a key that does not already exist.
