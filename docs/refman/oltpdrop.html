NAME

oltpdrop − Drop all oltp workload schemas

SYNOPSIS

oltpdrop

USAGE

The oltpdrop command that takes no options will drop all oltp schemas except the repository as defined by your parameter file. You may get errors if a previously executed oltpcreate has not been running successfully to completion.
